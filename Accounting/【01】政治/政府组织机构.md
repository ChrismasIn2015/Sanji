# 2.一些常见名词

```
1.经济单位
    企业单位：自负盈亏的生产性单位。
    事业单位：公益性质机构，予以财政补助
    行政单位：负责国家各项行政事务组织、管理指挥
2.法律等级
	宪法
    	- 法律（规范） - 法规（具体条例）
		- 规章（应情制定）
		- 地方条例/司法解释/国际条约协定
```
