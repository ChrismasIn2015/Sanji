使用途径

Bootstrap （当前版本 v3.3.7）提供以下几种方式帮你快速上手， 每一种方式针对具有不同技能等级的开发者和不同的使用场景。 继续阅读下面的内容，看看哪种方式适合你的需求吧。

用于生产环境的BootStrap

编译并压缩后的 CSS、JavaScript 和字体文件。不包含文档和源码文件。

下载BootStrap

Bootstrap 源码

Less、JavaScript 和 字体文件的源码，并且带有文档。 需要Less编译器和一些设置工作。

下载源码

CDN服务

使用 BootCDN 提供的免费 CDN 加速服务（同时支持 http 和 https 协议） Bootstrap 中文网 为 Bootstrap 专门构建了免费的 CDN 加速服务， 访问速度更快、加速效果更明显、没有速度和带宽限制、永久免费。 BootCDN 还对大量的前端开源工具库提供了 CDN 加速服务， 请进入BootCDN 主页查看更多可用的工具库。
